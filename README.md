# ESP32_S3_Samples

.NET nanoFramework 演示微雪的 ESP32-S3-Zero 使用

板子介绍：[中文](https://www.waveshare.net/wiki/ESP32-S3-Zero) | [English](https://www.waveshare.com/wiki/ESP32-S3-Zero)

因硬件的特异性，项目均以 `ESP32-S3-Zero` 为例，其他开发板请自行修改一些引脚等配置。

# 案例说明

1. OLED_SSD13XX： OLED，I2C显示屏
2. BoardButton：板载用户按钮
